
# <p align="center">🎙️ Gemini Studios: The Voice-Driven Canvas</p>


## 🌟 Project Description

**Gemini Studios** is a revolutionary AI art application that replaces traditional buttons, sliders, and prompt boxes with a **natural voice interface**. Designed as a "Multimodal Creative Orchestrator," it allows artists to simply speak their vision into existence.

Instead of navigating complex panels, you collaborate with an intelligent agent that manages a layered canvas, selects the best AI models (Flux, Stable Diffusion, DALL·E), and handles the technical heavy lifting—leaving you in a pure creative flow state.

> "Paint a vibrant sunset over a calm ocean, then add a lone sailboat silhouetted against the horizon."

---

## 🚀 Key Features

| Feature | Description | Emoji |
| --- | --- | --- |
| **Voice-First Control** | The default interface is your voice. No typing, just talking. | 🗣️ |
| **Layered Intelligence** | Uses different AI models for different layers (e.g., SD for backgrounds, Flux for subjects). | 📑 |
| **Manual Override** | A dedicated "Manual" toggle that grants the user full traditional control when needed. | 🔘 |
| **App-Wide Agency** | The agent doesn't just paint; it navigates settings, exports files, and manages your gallery. | 🤖 |
| **Collaborative Sync** | Real-time updates for teams working on the same canvas remotely. | 👥 |

## 🛠️ Technical Specifications

This project is built with a **Cloud-First, Low-Resource** philosophy. It is specifically optimized to provide high-end professional creative power on accessible hardware.

### **Hardware Optimization**

Developed and tested on:

* **Primary Workstation:** HP Laptop (1.5GHz Processor | 4GB RAM | 32GB Storage)
* **Mobile Testing:** Samsung A05 (4GB RAM)

### **The Tech Stack**

* **Brain:** Gemini 2.0 Multimodal Live API
* **Vision Engine:** Hugging Face Inference API (Flux.1, Stable Diffusion 3.5, SDXL Turbo)
* **State Management:** Real-time WebSocket synchronization
* **Frontend:** React with Tailwind CSS for ultra-lightweight UI rendering

## 📋 Installation & Instructions

### 1. Prerequisites

You will need your API keys from the following providers:

* **Google AI Studio** (For the Gemini Agent)
* **Hugging Face** (For the Image Generation Models)

### 2. Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/gemini-studios.git

```


2. Enter the directory and install dependencies:
```bash
npm install

```


3. Create a `.env` file and add your keys:
```env
VITE_GEMINI_API_KEY=your_key_here
VITE_HF_TOKEN=your_token_here

```



### 3. Running the App

```bash
npm run dev

```

Open the app in your browser, ensure your microphone is enabled, and simply say: *"Gemini, let's start a new project."*
