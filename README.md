## 🌟 The Vision
In 2026, creativity shouldn't be limited by hardware or technical complexity. **VoxArt Studio** bridges the gap by using the **Gemini 2.0 Multimodal Live API** as a "Creative Lead." The agent doesn't just execute prompts; it understands artistic intent, manages layers, and orchestrates a team of specialized AI models—all while running efficiently on everyday hardware.

## 🚀 Getting Started

### 1. Requirements
* A **Hugging Face Access Token** (Read permissions).
* **Google AI Studio API Key** (Gemini 2.0 Live enabled).
* Chrome or Edge browser (for Web Audio API support).

### 2. Quick Setup
1. Clone the repository and run `npm install`.
2. Open the **Integration Settings** in-app.
3. Paste your API keys into the secure vault.
4. Toggle **"Manual Control"** to **OFF** and say: *"Gemini, let's create a cinematic landscape."*
